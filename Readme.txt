Install dependencies:- npm install
Run the project:- npm start

API calls:
To view=(GET)localhost:3000/products [to get all the produects available in DB]

To create=(PUT)localhost:3000/products/create [to create a product with name and quantity]

To delete=(DELETE)localhost:3000/products/{id:} [to delete a product in id field just copy the genarated id]


To update=(PUT)localhost:3000/products/{:id}/update_quantity/?number={n} [to update a product value, put the id and updated value in n]


I have called those API with Thunder Client.
